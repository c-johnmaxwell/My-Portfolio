## 📱 Responsive Design

The portfolio is built with a mobile-first approach and includes breakpoints for:

- **Mobile**: 320px - 474px
- **Small Mobile**: 475px - 639px
- **Tablet**: 640px - 767px
- **Small Desktop**: 768px - 1023px
- **Large Desktop**: 1024px - 1279px
- **XL Desktop**: 1280px - 1535px
- **2XL Desktop**: 1536px+

### CSS Custom Properties
```css
:root {
  --size-xxs: 0.5rem;
  --size-xs: 0.75rem;
  --size-sm: 0.875rem;
  /* ... additional size variables */
}
```
